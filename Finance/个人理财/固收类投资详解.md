# 固收类投资详解 (Fixed-Income Investment)

固收类投资（固定收益类投资）是指投资于银行存款、债券、非标准化债权类资产等债权类资产的投资。其核心特征是**约定利率**或**预期收益相对稳定**，虽然在现代净值化转型后并非“保本保收益”，但其波动性通常显著低于权益类市场。

## 1. 核心特征

- **低波动性**：价格波动通常较小，适合作为资产配置的“压舱石”。
- **收益稳定性**：主要通过利息红利获取收益。
- **优先清偿权**：在企业破产清算时，债权人的受偿顺序优先于股东（权益类）。

## 2. 主要产品类型

### 2.1 现金及货币类（高流动性，极低风险）

- **银行存款**：
  - **活期/定期**：受《存款保险条例》保护（50万以内 100% 赔付），流动性好但收益偏低。
  - **大额存单**：起存金额通常为 20 万，利率通常高于普通定期，且支持在二级市场转让。
- **货币基金（MMF）**：
  - 代表产品如余额宝等。投资于极短期限的债权。
  - **优势**：门槛低（1元起）、流动性强（通常支持部分 T+0 快速赎回）。
  - **局限**：收益随全社会资金利率水平变动，目前普遍处于较低区间。

### 2.2 债券类（资产配置重心）

- **利率债**：
  - **国债/地方债**：以国家或地方政府信用为背书，几乎无违约风险。是市场的“避险资产”。
- **信用债**：
  - **金融债/企业债/公司债**：由银行或企业发行。收益包含“无风险利率 + 信用利差”。
  - **核心逻辑**：信用利差代表了市场对发行人违约概率的补偿，等级越高收益越低。
- **可转债（特殊存在）**：
  - **债性**：提供票面利息和到期本金收回保障（债底）。
  - **股性**：赋予投资者以固定价格转换成股票的权利。是典型的“下有保底，上有空间”的工具。

### 2.3 银行理财产品（净值化转型后）

- **固收型（通常为 R1-R2 级）**：
  - 80% 以上资产投向债权类。分为封闭式和开放式。
  - **风险等级说明**：
    - **R1（谨慎型）**：主要投向货币市场，极低风险。
    - **R2（稳健型）**：主要投向债券，不保本但波动极小，适合绝大多数风险保守者。
- **固收+（通常为 R2-R3 级）**：
  - 以固收资产打底，拿出 5%-20% 投资于权益类（股票/打新/衍生品）以博取超额收益。

### 2.4 信托/资管计划（高净值门槛）

- **投向**：房地产债权、基础设施债权、工商业贷款等。
- **特征**：起点通常 100 万起，期限较长（1-2年）。
- **现状**：随着监管“打破刚兑”，此类产品的非标债权正在减少，风险溢价显著。

## 3. 补充说明：进阶理解固收

### 3.1 什么是“净值化转型”？

过去固收产品多为“预期收益率”，即便底层亏了，机构也会垫付（刚性兑付）。
现在的产品均为**净值型**：盈亏自负，收益计算公式为 `(当前净值 - 初始净值) / 初始净值`。这意味着固收也会有“绿了”的时候（账面短期浮亏）。

### 3.2 债券价格与市场利率的关系

债券价格与市场利率呈**反向变动**：

- 当市场利率**上升**时，存量旧债券（利率相对变低了）吸引力下降，价格会**下跌**。
- 当市场利率**下降**时，存量旧债券（利率相对变高了）会变得抢手，价格会**上涨**。

### 3.3 信用评级简析

- **AAA**：信用极好（国企、头部金融机构）。
- **AA+ / AA**：信用优良。
- **AA- 以下**：通常被视为高收益债（也意味着高违约风险）。

### 3.4 标与非标：什么是非标准化债权？

在固收领域，你常会听到“标”与“非标”。

- **标准化债权资产（标）**：指在银行间市场、交易所市场交易的债权资产（如国债、企业债）。它们具备**等分化、可交易、信息披露充分、集中登记、流动性强**等特征。
- **非标准化债权资产（非标）**：简单理解就是“除此之外的所有债权”。通常是指投向房地产项目、地方融资平台或底层企业的**私募贷款、信托贷款、带回购安排的股权融资**等资产。

**为什么叫“非标准化”？**

1. **流动性差**：不像股票、基金能随时卖，非标通常要持有到期。
2. **定价不透明**：没有公开的市场交易价格。
3. **准入门槛高**：通常通过信托、资管计划发行，参与者多为银行理财或高净值投资者。
4. **曾是影子公司中心**：过去非标是影子银行主要资产，因为收益高且能通过嵌套躲避监管，但由于风险集中且透明度低，目前正受到严格监管并不断缩减。

### 3.5 揭秘“信托”与“资管”

在购买理财时，常会看到“XX信托计划”或“XX资管计划”，它们本质上是两种**法律关系与监管路径**。

#### 1. 什么是信托 (Trust)？

- **定义**：受托人（信托公司）根据委托人（你）的意愿，为受益人的利益，对财产进行管理或处分。
- **法律核心**：**财产独立性**。即信托财产既不属于你，也不属于信托公司，即使信托公司倒闭，这笔财产也不会被清算给信托公司的债权人（这就是所谓的“信托破产隔离”）。
- **主要角色**：信托公司（全国目前仅有 67 家，牌照极其珍贵，受金融监管总局直管）。

#### 2. 什么是资管 (Asset Management)？

- **定义**：全称“资产管理计划”。是一个更广泛的集合概念。
- **发送主体**：券商、基金公司子公司、保险资管、银行理财子公司等。
- **分类**：
  - **集合资管**：向多个客户募集资金。
  - **单一资管**：针对单一高净值或机构客户的“私人订制”。
- **实质**：资管更多是一种**买者自负**的契约关系。过去很多资管底层直接投向非标债权。

#### 3. 它们之间的联系与区别

- **联系**：两者都是典型的“受人之托，代人理财”。在过去，很多资管计划会去买信托计划（通道业务），目的是通过层层嵌套来规避监管。
- **区别**：
  - **牌照**：信托是全牌照，几乎什么都能投（债、股、贷、实物资产）；资管牌照受发售主体（如券商或基金）的限制，投向相对垂直。
  - **门槛**：信托通常 100 万起步（由于目前由于分类改革，部分小额信托门槛也在降低）；资管计划门槛通常也较高（30-100万起）。

## 4. 固收类投资的风险

1. **信用风险（违约风险）**：发行人无法按时兑付本息。
2. **利率风险**：市场利率上升时，存量债券价格通常会下跌。
3. **流动性风险**：在需要用钱时无法按合理价格快速变现。
4. **通胀风险**：收益率若低于通胀率，实际购买力会缩水。

## 4. 在资产配置中的作用

- **提供流动性**：满足短期开支和应急需求。
- **对冲波动**：在股市大跌时，固收类资产通常能保持稳定甚至小幅上涨。
- **获取确定性收益**：为未来的教育、养老等刚性支出提供现金流支撑。
